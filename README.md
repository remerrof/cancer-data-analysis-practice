# Paediatric Cancer Dataset Analysis Practice
## T-ALL Analysis based on Pölönen et al. 2024 paper
This repository contains Jupyter notebooks for analysing the dataset from the Pölönen et al. 2024 study. It aims to both recreate the analyses performed in the study, particularly clustering, and to do additional survival analysis.

The methods used are: 
1. Cox regression 
2. UMAP clustering
3. Survival analysis
4. Feature selection
